Measures of variation

Last time, we looked at how to measure the center of a data set.

Next, we want to look at how widely a data set is spread out.

Compare these data sets:

  • 5, 10, 15, 20, 25
  • 5, 6, 15, 24, 25

Both have the same mean (and median): 15.

But the second is “more spread out”.

We want to quantify this difference.

Range

The first way we measure variation is with the range.

The range of a data set is the difference between the maximum and minimum values:

  • range = max - min

For the data set 5, 10, 15, 20, 25, the range is 25 - 5 = 20.

For the data set 10, 40, 60, 90, the range is 80.

Standard deviation

The range does not help us see a difference in these data sets:

  • 5, 10, 15, 20, 25
  • 5, 6, 15, 24, 25

Both have a range of 20.

We want a measure of variation that uses all the points, not just the largest and smallest.

Standard deviation

The most common measure is called the standard deviation.

We will learn the process to compute it, but it is more often computed by software.

Computing standard deviation

First, make a table with one row per data value. Put in the data values and the mean of the data.

$x$$\bar x$
$5$$15$
$10$$15$
$15$$15$
$20$$15$
$25$$15$

Computing standard deviation

First, make a table with one row per data value. Put in the data values and the mean of the data.

Second, add a column for $x - \bar x$. This is the first column minus the second.

$x$$\bar x$$(x - \bar x)$
$5$$15$$-10$
$10$$15$$-5$
$15$$15$$0$
$20$$15$$5$
$25$$15$$10$

Computing standard deviation

First, make a table with one row per data value. Put in the data values and the mean of the data.

Second, add a column for $x - \bar x$. This is the first column minus the second.

Third, square each value from the third column.

$x$$\bar x$$(x - \bar x)$$(x-\bar x)^2$
$5$$15$$-10$$100$
$10$$15$$-5$$25$
$15$$15$$0$$0$
$20$$15$$5$$25$
$25$$15$$10$$100$

First, make a table with one row per data value. Put in the data values and the mean of the data.

Second, add a column for $x - \bar x$. This is the first column minus the second.

Third, square each value from the third column.

$x$$\bar x$$(x - \bar x)$$(x-\bar x)^2$
$5$$15$$-10$$100$
$10$$15$$-5$$25$
$15$$15$$0$$0$
$20$$15$$5$$25$
$25$$15$$10$$100$

Fourth, add up the final column: $100 + 25 + 0 + 25 + 100 = 250$.

Computing the standard deviation

Divide the sum you obtained by $N -1$. In our case, $N = 5$, so we compute \[ \frac{ 250}{4} = 62.5 \] Take the square root to get \[ \sqrt{62.5} = 7.91 \] This is the sample standard deviation.

Statistics notation

We can summarize the sample standard deviation in statistics notation.

First, we found each value of $(x - \bar x)^2$ and added them up. This is \[ \sum (x - \bar x)^2 \]

Then we divided by $N-1$. This is \[ \frac{\sum (x - \bar x)^2}{N-1} \]

Finally, the sample standard deviation is \[ s = \sqrt{\frac{\sum(x - \bar x)^2}{N-1}} \] We use an “s” to stand for the sample standard deviation.

Quick estimate for standard deviation

Rule of thumb estimate: \[ s \approx \dfrac{max - min}{4} \]

If your computed $s$ is more than your range, you should double check $s$.

What does standard deviation measure?

Data points make the standard deviation larger when they make $(x - \bar x)^2$ larger.

The farther a data point is from the mean, the bigger that square is.

The square root helps undo the squaring.

So in the end, we have something that measures a combination of how far each data point is from the mean.

Check your understanding

A data set for a continuous variable has mean 16.7, range 9, and sample standard deviation 3.6.

  • Does it make sense to compute the data value that is 1 "range" below the mean? If so find the value.
    • The range is the entire distance from the smallest value to the largest. The mean should be somewhere between these two values. It does not make sense to compute a value 1 "range" below the mean. This point does not exist (it wpuld be smaller than the minimum data value).
  • Does it make sense to compute the data value that is 1 "standard deviation" below the mean? If so find the value.
    • Using the rough estimate $s \approx (max - min) / 4$, likely the value that is 1 "standard deviation" below the mean is above the min (in the data set).
    • The value 1 "standard deviation" below the mean is $16.7 - 9 = 10.7$
    • Later on we'll do this type of calculation a lot. It will work well for data that is normally distributed.

Population standard devitation

There is a second kind of standard deviation called the population standard deviation.

To find this, we use the same procedure, but we divide by $N$ instead of $N-1$.

We use $\sigma$ to stand for the population standard deviation. This is a Greek lower case sigma.

Computing a population standard deviation

Find the population standard deviation of 74, 80, 90, 92.

The mean is $(70 + 80 + 90 + 92)/4 = 84$

Make a table:

$x$$\bar x$$(x - \bar x)$$(x - \bar x)^2$
$70$$84$$-14$$196$
$80$$84$$-4$$16$
$90$$84$$6$$36$
$92$$84$$8$$64$

Find $\sum (x - \bar x)^2 = (196 + 16 + 36 + 64) = 312$

Divide by $N$: $312/4 = 78$

Take a square root: \[ \sigma = \sqrt{78} = 8.83 \]

Which to use?

If we have data for an entire population, we use the population standard deviation.

If we have a sample, the sample standard deviation gives a better estimate of the standard deviation of the entire population (which we don't know).

Why? If we have a random sample, we are less likely to find data at the extremes of the population, because there are fewer population members at the extremes.

So the standard deviation of our sample tends to understate the standard deviation of the population. Dividing by $N-1$ helps counteract this.

The exact reason for using $N-1$ is covered in more advanced courses.

Coefficient of variation

The coefficient of variation is the standard deviation divided by the mean.

For example, if the standard deviation is $3$ and the mean is $12$, the coefficient of variation is $3/12 = 0.25$.

You cannot compare two standard deviations directly, because they may have different scales.

You can compare coefficients of variation to compare data sets with different scales.

Chebyshev’s Theorem

We can use standard deviations like a ruler. The mean is in the middle, and we count by standard deviations on each side.

Chebyshev’s Theorem is a result about that tells us how much data must be in certain regions on that ruler.

All that we need to know is the population mean, $\mu$, and the population standard deviation, $\sigma$.

We can also use the sample mean, $\bar x$, and the sample standard deviation, $s$.

Chebyshev's theorem

  • At least $75\%$ of the data falls within two standard deviations of the mean:
    • Between $\mu - 2\sigma$ and $\mu + 2\sigma$
    • Between $\bar x - 2s$ and $\bar x + 2s$
  • At least $88.9\%$ of the data falls within three standard deviations of the mean:
    • Between $\mu - 3\sigma$ and $\mu + 3\sigma$
    • Between $\bar x - 3s$ and $\bar x + 3s$
  • At least $93.8\%$ of the data falls within four standard deviations of the mean:
    • Between $\mu - 4\sigma$ and $\mu + 4\sigma$
    • Between $\bar x - 4s$ and $\bar x + 4s$

Example

Suppose we have a data set with a mean of $40$ and a standard deviation of $6$.

  • At least $75\%$ of the population will be between $28$ and $52$
  • At least $88.9\%$ of the population will be between $22$ and $58$
  • At least $93.8\%$ of the population will be between $16$ and $64$

It is possible for even more of the population to be in these regions. Chebyshev's theorem only gives a lower bound.

Goals of this lesson

  • Compute the range of a data set
  • Compute the sample standard deviation and population standard deviation of a data set
  • Understand the general difference between sample standard deviations and population standard deviations
  • Use Chebyshev’s Theorem to find intervals that contain at least $75\%$, $88.9\%$, or $93.8\%$ of the data.
