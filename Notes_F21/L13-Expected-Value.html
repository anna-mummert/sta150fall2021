discrete Random Variable

A discrete random variable is a random process that has a finite number of outcomes, each of which is a number.


Example

Roll a die and record the number that comes up.

For each class, record your grade earned.

Probability Distribution

The probabilities of all outcomes of an event have to add to 1.

The collection of all outcomes and their probabilities is called a discrete probability distribution.

Expected Value

The expected value is a weighted average of all the outcomes.

  • Multiply each outcome by its probability
  • Add up these products -- this is the expected value

The expected value is what we would get if we measured the random variable many times and then averaged our results.

Notation

$E(x)$ is the expected value of random variable $x$.

The formula for expected value is \[ E(x) = \sum x P(x) \]

Example

Find the expected number of pre-natal visits that a woman will make.

Data from a random sample of 50 mothers (from the Natality 2019 data) is given here.

Visits 1 4 5 6 7 8 9 10 11 12
Probability $\dfrac{1}{50}$ $\dfrac{2}{50}$ $\dfrac{7}{50}$ $\dfrac{8}{50}$ $\dfrac{13}{50}$ $\dfrac{9}{50}$ $\dfrac{3}{50}$ $\dfrac{2}{50}$ $\dfrac{3}{50}$ $\dfrac{2}{50}$

First notice that this is a valid probability distribtion because the probabilities add to 1.

Check your understanding

Determine if each of the following is a valid probability distribution. If it is, find the expeced value.

  • $x$ $-5$ $-3$ $2$ $10$
    $P(x)$ 0.2 0.5 0.3 0.2
    • This is not a valid probability distribution because $\sum P(a) = 1.2$. The probabilities must add to $1$.
  • $a$ $-5$ $-4$ $10$
    $P(a)$ 0.3 0.5 0.2
    • This is a valid probability distribution because $\sum P(a) = 1$.
    • The expected value is $-5*0.3 + -4*0.5 + 10*0.2 = -1.5$
